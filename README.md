

🖥️ LLM GPU Calculator
My first project — built with curiosity and a little help from AI.
While exploring the world of artificial intelligence, I stumbled upon Large Language Models (LLMs) and got hooked. One question kept bugging me: "How much GPU power do I actually need to run one of these models?" I couldn't find a simple, straightforward answer — so I decided to build one myself.
This tool helps you estimate the GPU/VRAM requirements needed to run various LLMs locally. Whether you're a curious beginner or someone planning to self-host a model, this calculator gives you a quick ballpark so you know what hardware you're working with before diving in.

✨ What it does

Estimates VRAM needed based on model size and configuration
Helps you figure out if your current GPU can handle a given LLM
Built for anyone who's ever wondered "can my machine run this?"


🚀 How it started
This is my first ever repository. I built it while learning about AI, LLMs, and how they work under the hood. It's not perfect, but it's mine and it solves a real problem I had.
